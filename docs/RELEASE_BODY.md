### English — Install

1. Download **WebDock-macOS-{{VERSION}}.zip** below  
2. Unzip → open **WebDock.app**  
3. System Settings → **Privacy & Security**  
   - **Screen Recording** → enable WebDock  
   - **Accessibility** → enable WebDock  
4. In the app: start the server (default port `8080`), set a token if you want  
5. Browser: `http://127.0.0.1:8080` (or your Mac’s LAN IP)  
6. Select a window and control it  

**Requirements:** macOS 14+, modern browser.  
**Source build:** see [README](../README.md) (`./install_home.sh`).
